Resumen de Temas Aprendidos

A lo largo del curso hemos explorado diversos conceptos fundamentales del desarrollo web frontend. Aquí está un resumen completo de todos los temas que hemos cubierto.

Diseño Adaptativo

Técnicas para crear sitios web que se adaptan a diferentes dispositivos y tamaños de pantalla.

  • Unidades relativas: rem, em, vh, vw, vmin, vmax
  • Tipografía responsiva con clamp() y calc()
  • Imágenes adaptativas con srcset y sizes
  • Elemento <picture> para dirección de arte
  • @font-face para tipografías personalizadas
  • Object-fit para control de imágenes
  • Mobile-first approach

Manipulación del DOM

Uso de JavaScript para interactuar dinámicamente con elementos HTML.

  • querySelector y querySelectorAll
  • createElement y appendChild
  • addEventListener para eventos
  • innerHTML, textContent y value
  • classList (add, remove, toggle)
  • setAttribute y getAttribute
  • Manipulación de estilos con style
  • Event delegation y bubbling

Media Queries

Aplicación de estilos CSS específicos según características del dispositivo.

  • Breakpoints estándar (móvil, tablet, desktop)
  • @media screen and (max-width: 768px)
  • Orientación: portrait y landscape
  • Grid y Flexbox responsivos
  • Menús adaptativos (hamburguesa)
  • Layouts que cambian según pantalla
  • prefers-color-scheme para modo oscuro

Animaciones CSS

Creación de efectos visuales dinámicos y transiciones suaves.

  • @keyframes para definir animaciones
  • animation: nombre duración timing-function
  • transition para cambios suaves
  • transform: translate, rotate, scale, skew
  • Efectos hover interactivos
  • Animation timing functions (ease, linear)
  • Animation delay e iteration count

Flexbox CSS

Sistema de layout flexible para diseños modernos y responsivos.

  • display: flex para activar Flexbox
  • justify-content para alineación horizontal
  • align-items para alineación vertical
  • flex-direction: row | column
  • flex-wrap para múltiples líneas
  • gap para espaciado entre elementos
  • flex: grow shrink basis
  • order para cambiar orden visual

HTML5 y CSS3

Estructura semántica y estilos modernos para la web.

  • Etiquetas semánticas (header, nav, main, footer)
  • Variables CSS con :root
  • Box-sizing: border-box
  • Position: fixed, sticky, absolute
  • Border-radius, box-shadow
  • Pseudo-clases (:hover, :focus, :active)
  • Formularios accesibles con labels
  • Gradientes lineales y radiales
