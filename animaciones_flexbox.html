Animaciones y Flexbox CSS

Explorando animaciones, transiciones, transformaciones y el poder de Flexbox para layouts modernos

Animaciones CSS

Las animaciones CSS permiten crear efectos visuales dinámicos sin necesidad de JavaScript. Utilizando @keyframes, transitions y transforms, podemos dar vida a nuestras interfaces web, mejorar la experiencia de usuario y crear interacciones atractivas y profesionales.

Código

/* animación con @keyframes */
@keyframes bounce {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
}

/* Aplicar animación */
.elemento {
    animation: bounce 1s infinite;
}

/* Transiciones suaves */
.boton {
    transition: all 0.3s ease;
}

.boton:hover {
    transform: scale(1.1);
    background: blue;
}

Conceptos Clave:

  • @keyframes: Define los estados de la animación
  • animation: Aplica la animación al elemento
  • transition: Suaviza cambios de propiedades CSS
  • transform: Aplica transformaciones 2D/3D (translate, rotate, scale, skew)
  • timing-function: Controla la velocidad (ease, linear, ease-in-out)

Flexbox CSS

Flexbox (Flexible Box Layout) es un modelo de diseño CSS que facilita la creación de layouts flexibles y responsivo. Permite distribuir espacio y alinear elementos de manera eficiente, incluso cuando sus tamaños son desconocidos o dinámicos.

Código de Flexbox

/* Contenedor Flex básico */
.contenedor {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 1rem;
}

/* Layout con flex-direction */
.menu-lateral {
    display: flex;
    flex-direction: column;
}

/* responsivo con flex-wrap */
.galeria {
    display: flex;
    flex-wrap: wrap;
    gap: 2rem;
}

/* Items flexibles */
.item {
    flex: 1 1 300px; /* grow, shrink, basis */
}

Propiedades Principales de Flexbox:

  • display: flex; Activa el modelo Flexbox en el contenedor
  • flex-direction: Define la dirección (row, column, row-reverse, column-reverse)
  • justify-content: Alineación en el eje principal
  • align-items: Alineación en el eje transversal
  • flex-wrap: Permite que los items se ajusten en múltiples líneas
  • gap: Define el espacio entre elementos flex
  • flex: Propiedad abreviada para flex-grow, flex-shrink y flex-basis
