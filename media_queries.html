Diseño Responsivo con Media Queries

Demostración de diferentes técnicas de diseño adaptativo para múltiples dispositivos

¿Qué son las Media Queries?

Las Media Queries son una característica de CSS3 que permite aplicar estilos específicos según las características del dispositivo, como el ancho de pantalla, la orientación, la resolución, etc. Son fundamentales para crear diseños responsivos.

/* Ejemplo básico de Media Query */
@media screen and (max-width: 768px) {
    .contenedor {
        flex-direction: column;
        padding: 1rem;
    }
}

Breakpoints Utilizados

Estos son los puntos de quiebre implementados en esta página:

Desktop (>1024px)

  • Menú lateral fijo
  • Grid de 4 columnas
  • Cards en 3 columnas
  • Sidebar visible
  • Efectos hover completos

Tablet (481-1024px)

  • Menú superior horizontal
  • Grid de 2 columnas
  • Cards en 2 columnas
  • Menú hamburguesa
  • Layout adaptado

Celular (≤480px)

  • Menú vertical superior
  • Grid de 1 columna
  • Cards en 1 columna
  • Sidebar debajo
  • Optimizado táctil

Código de Media Queries

/* Desktop Grande */
@media screen and (min-width: 1200px) {
    .grid { grid-template-columns: repeat(4, 1fr); }
}

/* Tablet */
@media screen and (max-width: 1024px) {
    .sidebar {
        position: fixed;
        top: 0;
        transform: translateY(-100%);
    }
    .grid { grid-template-columns: repeat(2, 1fr); }
}

/* Celular */
@media screen and (max-width: 768px) {
    .grid { grid-template-columns: 1fr; }
    .layout { flex-direction: column; }
}

/* Orientación */
@media (orientation: landscape) {
    .hero { height: 100vh; }
}
