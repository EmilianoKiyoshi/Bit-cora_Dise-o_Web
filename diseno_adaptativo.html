Unidades de Medida CSS

Demostracion practica de diferentes unidades de medida y sus casos de uso especificos:

Titulo usando REM

Este titulo usa 2rem. Las unidades rem son relativas al elemento raiz (html), lo que las hace predecibles y escalables. Si el root font-size es 16px, este titulo sera 32px. Ideal para titulos porque escalan uniformemente.

Uso: Titulos, elementos que necesitan escalado consistente

Subtitulo usando EM

Este texto usa 1.1em. Las unidades em son relativas al elemento padre, por lo que escalan proporcionalmente. Si el padre es 16px, este texto sera 17.6px.

Uso: Texto que debe escalar con su contexto, componentes anidados

Bloque con altura VH

Este contenedor usa 30vh (30% del viewport height). Las unidades vh son ideales para secciones que deben ocupar una porcion especifica de la pantalla visible.

Uso: Heroes, secciones de pantalla completa, elementos que dependen de la altura

Elemento con ancho VW

Este elemento usa 80vw (80% del viewport width). Se adapta automaticamente al ancho de pantalla disponible.

Uso: Elementos fluidos, contenedores que deben adaptarse al ancho

Texto VMIN

Font-size de 4vmin. Se basa en la dimension mas peque√±a del viewport (ancho o alto), garantizando legibilidad en cualquier orientacion.

Uso: Texto que debe mantenerse legible en cualquier orientacion

Texto VMAX

Font-size de 3vmax. Se basa en la dimension mas grande del viewport, util para elementos decorativos que deben ser prominentes.

Uso: Elementos decorativos, titulos de impacto visual

Ejemplos de Codigo Implementado

Unidades Responsivas

/* REM - relativo al root */
.titulo-rem { font-size: 2rem; }

/* EM - relativo al padre */
.subtitulo-em { font-size: 1.5em; }

/* VH - porcentaje de altura */
.bloque-vh { height: 30vh; }

/* VW - porcentaje de ancho */
.bloque-vw { width: 80vw; }

Tipografia Responsiva

/* clamp() - automatico */
font-size: clamp(1rem, 2.5vw, 1.4rem);

/* calc() - combinado */
font-size: calc(1rem + 0.5vw);

/* Media queries condicionales */
@media (max-width: 480px) {
  font-size: 0.9rem;
}
